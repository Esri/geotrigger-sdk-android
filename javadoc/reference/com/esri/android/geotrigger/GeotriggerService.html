Class Overview

A background Service that records locations, periodically posting them to the Geotrigger API.

You can instantiate and register a GeotriggerBroadcastReceiver and implement its interfaces on your Activity or Service in order to receive callbacks when certain events occur.

public static void setLegacyTrackingModeFallbackEnabled (Context context, boolean enabled)

Allow the GeotriggerService to fall back to legacy LocationManager based tracking.

This option should not be enabled unless your app needs to run on devices that are not supported by Google Play Services, as the legacy tracking method will drain the battery much more quickly than the default LocationClient based tracking method.

Parameters
context a Context object.
enabled true to enable fallback to legacy tracking.

public static void setLoggingLevel (int level)

Set the log level to be used by the Geotrigger SDK.

The default value is Log.WARN for release builds and Log.DEBUG for debug builds. Log.VERBOSE will show additional logs for HTTP requests and other events, but should not be used for released applications.
